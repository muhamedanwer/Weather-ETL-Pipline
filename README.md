## Weather ETL Pipeline


- This pipeline will extract weather data about the last 3 months from the open-meteo API
- The data will be stored in a Postgres database.
- The data will be used to create a dashboard with Streamlit.


#### Tools:
- Python
- Postgres
- Streamlit
- Docker
- Airflow


### Customization

    - Add more cities in config.py
    - Modify transformation logic in transform.py
    - Create custom visualizations in Jupyter notebook
